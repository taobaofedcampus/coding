输出：一条直线能够连接最多点的数量

实现：getMaxPointsLength

---

注意：

示例如下：
```javascript
  const points=[[0, 0, 0], [1, 1, 1], [2, 2, 2], [0, 0, 3]]

  getMaxPointsLength(points);   //3，连接的点分别是[0, 0, 0], [1, 1, 1], [2, 2, 2]
```
